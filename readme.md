## 初始程序加载
1. 引擎查找并加载 `emuera.config` 中的配置信息（窗口、字体、语言、渲染设置等）。
2. 引擎扫描工程目录下的脚本目录（本工程为 `erb/`），按加载顺序解析并执行 `.erb` 脚本文件。
3. 若项目使用 `csv/` 数据表，脚本中会在运行时读取这些表以初始化角色、物品、地图或对话数据。
4. 引擎在运行期间会写入日志文件（例如 `emuera.log`、`time.log`）以及存档文件（`profile` 或 `SAV/` 下的保存数据）。
5. 启动流程的典型步骤：
	- 启动可执行文件（例如 `Emuera.NET ...exe`）。
	- 引擎读取 `emuera.config`，并根据配置设置窗口与语言。
	- 扫描 `erb/` 并执行入口脚本（若存在类似 `main.erb` 或首个脚本文件）。

提示：运行前建议备份 `profile`（二进制存档），并确认 `.gitignore` 已正确配置以避免将大型二进制与日志提交到仓库。
